# Project: Student Reality Lab

## Commits

- Use conventional commit style: `type(scope): description`
- Never add "Co-authored-by: Claude", "Generated by Claude", or any AI attribution
- Commit after each major task/phase milestone

## Stack

- Next.js 15 (App Router), React 19, TypeScript strict mode
- Bun as runtime and package manager (use `bun` not `npm`/`yarn`)
- Tailwind CSS v4, Recharts for charts, Zod for validation
- Vitest for tests

## Code conventions

- All data loading happens server-side at build time (no client-side fetch for data)
- Charts are lazy-loaded with `dynamic(() => import(...), { ssr: false })`
- Use `src/utils/cn.ts` for className merging
- Validate all external data with Zod schemas in `src/lib/data/schema.ts`

## Design system

- Swiss International style: Inter font, 8px grid, minimal palette
- Colors: ink `#0a0a0a`, paper `#fafafa`, 4yr path `#2563eb`, transfer `#ea580c`
- Max content width 960px, 64px+ between sections
